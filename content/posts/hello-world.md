# 欢迎来到我的博客！

这是使用 **Next.js 14** 重构的 Gal 主题博客的第一篇文章。

## 技术栈

本博客采用以下技术栈：

- **Next.js 14** - React 框架，使用 App Router
- **TailwindCSS** - 原子化 CSS 框架
- **shadcn/ui** - 精美的 UI 组件库
- **Lucide Icons** - 现代图标库

## 特性

### 🎨 精美的 UI 设计

保留了原 Gal 主题的设计风格，同时使用现代化的技术栈重新实现。

### 🚀 极致的性能

得益于 Next.js 的静态生成和服务端渲染，博客加载速度极快。

### 📱 响应式设计

完美适配桌面端和移动端设备。
